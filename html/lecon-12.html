Leçon 12 : CSS

Responsive design

Le responsive design rend le site utilisable sur mobile, tablette et ordinateur. On adapte les styles en fonction de la largeur de l'écran pour garder un bon rendu partout.

Les media queries

Les media queries permettent de changer les styles selon la largeur de l'écran. C'est l'outil principal pour ajuster une mise en page responsive. 

.page {
  padding: 24px;
}

@media screen and (max-width: 768px) {
  .page {
    padding: 16px;
  }
}

Sur grand écran : plus d'espace autour du contenu.

Sur mobile, on réduit les marges pour maximiser l'espace de lecture.

Changer la disposition

Il est courant de passer d'une disposition en lignes à une disposition en colonnes sur les petits écrans pour éviter les éléments trop serrés. Ici on applique flex-direction: column dans une media query pour empiler les éléments. 

.bloc {
  display: flex;
  gap: 18px;
}

@media screen and (max-width: 720px) {
  .bloc {
    flex-direction: column;
  }
}

Texte et images adaptatives

Les images et les textes doivent s'adapter aux différents écrans. Utilise max-width: 100% pour les images et des unités relatives pour le texte. 

img {
  max-width: 100%;
  height: auto;
}

.titre {
  font-size: clamp(1.4rem, 2vw, 2.5rem);
}

Titre adaptatif

Prioriser le contenu

Sur mobile, on masque ou réorganise certains éléments pour clarifier la page. Le bloc le plus important doit rester visible en premier. 

.aside {
  display: block;
}

@media screen and (max-width: 640px) {
  .aside {
    display: none;
  }
}
Contenu principal
Contenu secondaire

Le display: none permet de masquer un élément sur les petits écrans pour alléger la page.

Points clés à retenir

  • Les media queries adaptent le style selon la taille d'écran.
  • Passer de lignes à colonnes aide les petits écrans.
  • Les images responsives utilisent max-width: 100%.
  • Sur mobile, priorise le contenu le plus important.
