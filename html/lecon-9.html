Leçon 09 · CSS

Le Box Model

C'est le concept fondamental du CSS. Tout élément HTML est une boîte rectangulaire. Cette boîte est composée de 4 couches. Comprendre le box model, c'est comprendre comment tout se positionne sur une page web.

Les 4 couches de la boîte

De l'intérieur vers l'extérieur : le contenu, le padding, la bordure, et la marge. Chaque couche a un rôle précis. 

.carte {
  /* 1. Le contenu : largeur et hauteur */
  width: 200px;
  height: 80px;

  /* 2. Padding : espace INTÉRIEUR
        entre le contenu et la bordure */
  padding: 20px;

  /* 3. Border : la bordure */
  border: 2px solid #333;

  /* 4. Margin : espace EXTÉRIEUR
        entre cet élément et les autres */
  margin: 30px;
}

Padding vs Margin : la différence clé

C'est la confusion la plus courante. Retiens ceci : le padding est à l'intérieur de la boîte (espace entre le texte et la bordure), la margin est à l'extérieur (espace entre deux éléments). 

/* Padding : espace intérieur */
.btn {
  padding: 10px 20px;
  /* 10px haut/bas, 20px gauche/droite */
  background: orange;
}

/* Margin : espace extérieur */
.carte {
  margin: 0 auto;
  /* 0 haut/bas, auto gauche/droite
     → centre l'élément horizontalement */
  max-width: 600px;
}

box-sizing : le piège à éviter

Par défaut, quand tu définis width: 200px, le padding et la bordure s'ajoutent à cette largeur — l'élément fait donc plus de 200px. Avec box-sizing: border-box, le padding et la bordure sont inclus dans la largeur. C'est pourquoi on l'applique toujours en premier dans le reset CSS. 

/* Comportement par défaut (piège) */
.boite {
  width: 200px;
  padding: 20px;
  /* Taille réelle = 200 + 40 = 240px ❌ */
}

/* Avec border-box (bonne pratique) */
* {
  box-sizing: border-box;
}
.boite {
  width: 200px;
  padding: 20px;
  /* Taille réelle = 200px ✅ */
}

La notation raccourcie

padding et margin acceptent 1, 2, 3 ou 4 valeurs. C'est un raccourci très utilisé pour éviter d'écrire quatre propriétés séparées. 

/* 1 valeur : tous les côtés */
padding: 20px;

/* 2 valeurs : haut/bas  gauche/droite */
padding: 10px 20px;

/* 3 valeurs : haut  gauche/droite  bas */
padding: 10px 20px 30px;

/* 4 valeurs : haut  droite  bas  gauche
   (sens des aiguilles d'une montre) */
padding: 10px 20px 30px 40px;

/* Côtés individuels */
padding-top: 10px;
padding-right: 20px;
margin-bottom: 30px;
margin-left: 40px;
padding: 8px → uniforme
padding: 4px 20px → plat et large
padding: 16px 8px 4px → haut lourd

Mémo : sens ↑ → ↓ ← (comme une montre)

Points clés à retenir

  • Tout élément HTML est une boîte : contenu → padding → border → margin
  • padding = espace intérieur (entre le texte et la bordure)
  • margin = espace extérieur (entre deux éléments)
  • Toujours déclarer * { box-sizing: border-box; } en début de CSS
  • margin: 0 auto sur un bloc avec max-width le centre horizontalement
