Leçon 10 · CSS

Flexbox

Flexbox est la méthode moderne pour aligner et répartir des éléments dans un conteneur. Une fois les notions de conteneur et d'éléments flex comprises, la mise en page devient beaucoup plus simple.

Le conteneur flex

Pour activer Flexbox, on transforme un parent en conteneur flex avec display: flex. Ensuite, les enfants deviennent des éléments flex. 

.barre {
  display: flex;
  gap: 16px;
  padding: 16px;
  background: #f8fafc;
}

.barre > div {
  background: #38bdf8;
  color: white;
  padding: 12px 18px;
  border-radius: 8px;
}

Direction et alignement

flex-direction choisit l'axe principal (ligne ou colonne). justify-content place les éléments sur l'axe principal, et align-items les aligne sur l'axe secondaire. 

.menu {
  display: flex;
  flex-direction: column;
  gap: 12px;
  align-items: center;
  justify-content: space-between;
  height: 240px;
}

La taille flexible

flex permet à un élément de grandir, rapetisser ou partir d'une valeur de base. Utilisé souvent avec flex: 1 pour répartir l'espace disponible. 

.cartes {
  display: flex;
  gap: 16px;
}

.carte {
  flex: 1;
  padding: 18px;
  background: #dbeafe;
  border-radius: 10px;
}

Changer l'ordre

Avec order, on peut réorganiser les éléments sans modifier l'HTML. Cela reste très pratique pour les mises en page responsive. 

.ligne {
  display: flex;
  gap: 12px;
}

.item-1 { order: 2; }
.item-2 { order: 1; }

Points clés à retenir

  • display: flex transforme un parent en conteneur flexible.
  • justify-content gère l'axe principal, align-items l'axe secondaire.
  • flex répartit l'espace entre plusieurs éléments.
  • order change l'ordre visuel sans toucher à l'HTML.
