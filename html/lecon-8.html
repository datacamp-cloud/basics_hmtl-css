Leçon 08 · CSS

Couleurs et polices

Les couleurs et la typographie sont les deux éléments qui donnent l'identité visuelle d'un site. Maîtriser ces propriétés CSS te permet de reproduire n'importe quelle charte graphique.

Les formats de couleur

CSS accepte plusieurs façons d'écrire une couleur. On utilise surtout l'hexadécimal et RGB dans la pratique. Pour les transparences, on utilise rgba(). 

p {
  /* Nom de couleur (limité) */
  color: orange;

  /* Hexadécimal : #RRVVBB */
  color: #f97316;

  /* RGB : rouge, vert, bleu (0-255) */
  color: rgb(249, 115, 22);

  /* RGBA : avec transparence (0 à 1) */
  color: rgba(249, 115, 22, 0.5);

  /* Fond de l'élément */
  background-color: #f8fafc;
}

Choisir une police : Google Fonts

Par défaut le navigateur affiche une police système basique. Google Fonts propose des milliers de polices gratuites qu'on importe dans le <head> du HTML, puis on les applique en CSS avec font-family. 

<!-- Dans le <head> du HTML -->
<link
  href="https://fonts.googleapis.com/
  css2?family=Playfair+Display:wght@700
  &family=Outfit:wght@400;500"
  rel="stylesheet">

/* Dans le CSS */
h1 {
  font-family: 'Playfair Display', serif;
}
p {
  font-family: 'Outfit', sans-serif;
}

Les propriétés typographiques

CSS propose de nombreuses propriétés pour contrôler l'apparence du texte. Voici les plus utilisées. 

p {
  /* Taille : px, rem, em, % */
  font-size: 1rem;

  /* Graisse : 400=normal, 700=gras */
  font-weight: 400;

  /* Hauteur de ligne */
  line-height: 1.6;

  /* Espacement entre lettres */
  letter-spacing: 0.05em;

  /* Alignement */
  text-align: center;

  /* Transformation */
  text-transform: uppercase;
}

Les variables CSS — bonne pratique

Plutôt que de répéter #f97316 partout dans ton CSS, on centralise les couleurs et polices dans des variables. Si on veut changer une couleur, on modifie un seul endroit. 

/* On déclare les variables sur :root
   (s'applique à toute la page) */
:root {
  --couleur-accent: #f97316;
  --couleur-texte: #1e293b;
  --police-titre: 'Playfair Display', serif;
}

/* On les utilise avec var() */
h1 {
  font-family: var(--police-titre);
  color: var(--couleur-accent);
}

Avantage des variables :

Tu changes --couleur-accent une fois...

...et tout le site change de couleur automatiquement.

Points clés à retenir

  • Préfère le format hexadécimal pour les couleurs et rgba() pour les transparences
  • Google Fonts s'importe dans le <head> avant le lien vers le CSS
  • Toujours déclarer une police de secours : 'Outfit', sans-serif
  • line-height: 1.6 améliore grandement la lisibilité du texte
  • Centralise tes couleurs et polices dans des variables CSS sur :root
