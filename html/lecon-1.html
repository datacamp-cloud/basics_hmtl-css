Leçon 01 · HTML

Structure d'une page HTML

Avant d'écrire quoi que ce soit, il faut comprendre le squelette minimal que tout fichier HTML doit contenir. C'est la fondation sur laquelle tout le reste s'appuie.

C'est quoi HTML ?

HTML signifie HyperText Markup Language. C'est le langage qui structure le contenu d'une page web. Il ne s'occupe pas des couleurs ou des positions — ça, c'est le rôle du CSS. HTML, c'est uniquement la structure et le sens du contenu.

Un fichier HTML est composé de balises. Une balise s'ouvre avec <nom> et se ferme avec </nom>. Tout ce qui est entre les deux est le contenu de cette balise.

Le squelette minimal d'une page

Voici le code minimum que tout fichier HTML doit contenir. À gauche le code, à droite ce que le navigateur retient. 

<!-- Déclare que c'est du HTML5 -->
<!DOCTYPE html>

<!-- Racine de toute la page -->
<html lang="fr">

  <!-- Infos invisibles pour le navigateur -->
  <head>
    <meta charset="UTF-8">
    <title>Mon site</title>
  </head>

  <!-- Contenu visible de la page -->
  <body>
    <p>Bonjour le monde !</p>
  </body>

</html>

Onglet du navigateur :

📄 Mon site

Contenu affiché :

Bonjour le monde !

Le rôle de chaque partie

Voici ce que fait chaque élément du squelette :

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport"
    content="width=device-width">
  <link rel="stylesheet"
    href="style.css">
  <title>Mon site</title>
</head>
<body>
  <!-- Tout le contenu visible ici -->
</body>
</html>
!DOCTYPE

Indique au navigateur que c'est du HTML version 5.

html lang

Définit la langue de la page (important pour l'accessibilité).

head

Contient les informations techniques invisibles à l'écran.

charset

Permet d'utiliser les accents français (é, è, à...).

body

Tout ce que l'utilisateur voit dans le navigateur.

Points clés à retenir

  • Tout fichier HTML commence par <!DOCTYPE html>
  • La balise <head> contient les infos techniques (invisible à l'écran)
  • La balise <body> contient tout ce que l'utilisateur voit
  • On lie le fichier CSS avec <link rel="stylesheet" href="style.css"> dans le <head>
  • Toujours ajouter lang="fr" et charset="UTF-8" pour les accents
