Les formulaires

Les formulaires permettent à l'utilisateur d'envoyer des données : s'inscrire, se connecter, contacter, rechercher... C'est la principale façon d'interagir avec un site web.

La balise <form>

Tout formulaire est enveloppé dans une balise <form>. Elle a deux attributs importants : action (où envoyer les données) et method (comment les envoyer — get ou post). 

<!-- action = URL de traitement des données -->
<!-- method = "post" pour envoyer des données -->
<form action="/envoyer"
      method="post">

  <!-- Les champs vont ici -->

</form>

Les deux méthodes :

GET

Données visibles dans l'URL — pour les recherches

POST

Données cachées — pour les formulaires de contact

Le duo <label> + <input>

Chaque champ de saisie est accompagné d'un <label> qui le décrit. Le lien entre les deux se fait via for sur le label et id sur l'input — ils doivent avoir la même valeur. Cliquer sur le label active automatiquement le champ correspondant. 

<!-- for="nom" ↔ id="nom" : ils sont liés -->
<label for="nom">Votre nom</label>
<input
  type="text"
  id="nom"
  name="nom"
  placeholder="Jean Dupont"
  required>

Les différents types d'<input>

L'attribut type change complètement le comportement du champ. Le navigateur adapte automatiquement le clavier sur mobile et la validation des données. Il en existe de nombreux types : text, email, password, phone, checkbox, radio... 

<!-- Texte libre -->
<input type="text">

<!-- Email : valide le format -->
<input type="email">

<!-- Mot de passe : masque les caractères -->
<input type="password">

<!-- Téléphone -->
<input type="phone">

<!-- Case à cocher -->
<input type="checkbox">

<!-- Bouton radio -->
<input type="radio">

type="text"

<select>, <textarea> et <button>

Ces trois éléments complètent la palette des champs de formulaire. <select> crée une liste déroulante, <textarea> un champ de texte multiligne, et <button type="submit"> envoie le formulaire. 

<!-- Liste déroulante -->
<select name="sujet">
  <option value="">Choisir...</option>
  <option value="projet">Projet</option>
  <option value="info">Information</option>
</select>

<!-- Zone de texte multiligne -->
<textarea
  name="message"
  rows="4"
  placeholder="Votre message...">
</textarea>

<!-- Bouton d'envoi -->
<button type="submit">
  Envoyer
</button>

Points clés à retenir

  • Tout formulaire est dans un <form> avec action et method
  • Le for du label doit correspondre à l'id de l'input
  • type="email" valide automatiquement le format — pas besoin de JavaScript
  • required rend un champ obligatoire
  • <button type="submit"> envoie le formulaire — c'est son rôle par défaut
