Leçon 11 : CSS

Mise en page

Cette leçon montre comment structurer un site avec une vraie navbar et un hero. On assemble les éléments CSS vus précédemment pour obtenir une page claire et moderne.

Créer une navbar simple

Une navigation se compose d'un conteneur et de liens alignés sur une ligne. Flexbox est parfait pour espacer les éléments et aligner le logo et les liens. 

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 18px 24px;
}

.nav-links {
  display: flex;
  gap: 18px;
}

.nav-links a {
  text-decoration: none;
  color: #0f172a;
}

Construire un hero

Le hero est la première section visible. Il combine un titre fort, un texte clair et un appel à l'action. On peut aussi utiliser un conteneur centré avec une largeur maximale pour améliorer la lecture. 

.hero {
  max-width: 760px;
  margin: 0 auto;
  padding: 60px 24px;
  text-align: center;
}

.btn-primary {
  display: inline-block;
  padding: 12px 24px;
  background: #2563eb;
  color: white;
  border-radius: 10px;
}

Construire une page claire et moderne

Utilise une structure simple, des sections espacées et un appel à l'action visible pour guider l'utilisateur.

Disposer plusieurs colonnes

On peut présenter plusieurs cartes côte à côte en utilisant Flexbox ou Grid. Ici, chaque carte a une largeur flexible et un espace régulier entre elles. 

.grille {
  display: flex;
  gap: 20px;
}

.carte {
  flex: 1;
  padding: 20px;
  background: #f1f5f9;
  border-radius: 14px;
}

Limiter la largeur du contenu

Un contenu trop large est difficile à lire. On utilise max-width et margin: 0 auto pour centrer proprement les sections et garder le texte confortable. 

.contenu {
  max-width: 900px;
  margin: 0 auto;
  padding: 0 24px;
}

Le contenu est plus facile à lire lorsque la ligne ne fait pas trop de largeur. Cette technique rend la page plus agréable sur grand écran.

Points clés à retenir

  • Une navbar utilise souvent display: flex pour aligner les éléments.
  • Le hero doit être clair, centré et attractif.
  • Les colonnes peuvent être créées avec Flexbox et des cartes flexibles.
  • max-width et margin: 0 auto améliorent la lisibilité.
