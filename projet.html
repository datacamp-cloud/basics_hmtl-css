Projet final

Construis un site complet

Après les 12 leçons, c'est le moment d'assembler tes compétences. Tu vas créer un site web fonctionnel, responsive et visuellement attractif.

Objectif du projet

Ce site est un modèle portfolio : il montre une structure simple et complète avec une navigation, un hero, des services, une section à propos et un formulaire de contact. Tu peux modifier les informations, les titres et les services pour en faire ton propre projet, mais suis d'abord les étapes recommandées.

  • Une navbar claire avec des liens.
  • Un hero fort avec un titre, un texte et un bouton.
  • Des cartes de services alignées avec Flexbox.
  • Une section à propos et un formulaire de contact.
  • Une mise en page responsive qui s'adapte au mobile.

Comment construire ce projet

Je te conseille de faire le projet en 3 étapes : structure, style, puis responsive. Commence par poser le HTML, puis ajoute des styles simples, et enfin adapte ton site aux petits écrans. 

header {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 20px 24px;
}

.services {
  display: flex;
  gap: 18px;
  flex-wrap: wrap;
}

img {
  max-width: 100%;
  height: auto;
}

Structure → Style → Responsive

C'est un modèle à personnaliser : remplace les textes et les services par tes propres idées, mais garde le bon ordre de travail.

Étapes pour reproduire ce portfolio

  1. Crée le squelette HTML avec les sections suivantes : header, .hero, .services, .apropos, .contact, et footer.
  2. Ajoute les classes de structure importantes : .nav-container, .services-grille, .carte-service, .apropos-layout, .contact-layout.
  3. Rédige le CSS de base : couleurs, typographie, marges/paddings, bordures et arrière-plans.
  4. Utilise Flexbox pour aligner la barre de navigation, la grille de services et les colonnes de la section à propos et contact.
  5. Ajoute un style de base pour les boutons, les cartes et le formulaire afin que le rendu soit propre et cohérent.
  6. Crée des media queries pour les écrans plus petits : empile les sections et affiche le menu mobile.
  7. Teste ton site dans le navigateur, puis compare avec le modèle et personnalise les textes et services.

Conseils pour réussir

  • Commence par une version mobile d'abord si tu veux.
  • Utilise des noms de classes simples : .navbar, .hero, .card.
  • Teste ton site dans le navigateur et redimensionne la fenêtre.
  • Relis chaque section de ton HTML pour vérifier la sémantique.
  • Sois libre sur le contenu et les couleurs : le projet est aussi personnel.

Ce que je te recommande

Ne te contente pas de copier la page finale. Essaie d'abord de faire ta propre version puis compare avec un exemple. Le plus important est de comprendre pourquoi tu fais chaque choix.

Plan conseillé

  • 1. Créer le HTML et la structure du site.
  • 2. Ajouter les styles de base, couleurs et typographie.
  • 3. Construire la mise en page avec Flexbox.
  • 4. Ajouter les media queries pour mobile.
  • 5. Vérifier les liens, images et formulaire.
